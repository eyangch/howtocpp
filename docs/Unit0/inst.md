# Installing
- Windows:
    - [TDM-GCC](http://tdm-gcc.tdragon.net/) (suggested)
    - [MinGW](http://www.mingw.org/)
    - Download and follow installer instructions.
- Linux (Debian based):
    - To install a C++ compiler (g++) on a Linux (Debian based) machine, run the following command.
    ```bash
    sudo apt install g++
    ```
